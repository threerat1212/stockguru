# StockGuru Safe Generic Finance Wording

Use this wording bank for StockGuru UI copy, AI responses, prompts, explainers, empty states, paywall copy, alerts, news summaries, War Room, MiroFish, SEGA, and trading journal feedback.

This is generic finance language adapted to StockGuru's safety rules. It must never become investment advice.

## Core Voice

- Thai-first, concise, evidence-led.
- Explain what happened, why it matters, what to verify next, and what could go wrong.
- Use probabilities, conditions, and uncertainty when discussing markets.
- Avoid commands that tell the user what to buy, sell, hold, or allocate.

## Safe Thai Phrases

### Market Context

- ภาพรวมตลาดวันนี้สะท้อนแรงซื้อขายที่กระจุกตัวในบางกลุ่ม
- ตลาดยังต้องการปัจจัยยืนยันจากมูลค่าซื้อขายและ fund flow
- ภาวะตลาดยังผันผวน ควรแยกข่าวระยะสั้นออกจากแนวโน้มหลัก
- ดัชนีเริ่มมีแรงฟื้นตัว แต่ยังควรตรวจแนวรับ แนวต้าน และ volume ประกอบ
- การเคลื่อนไหวนี้เป็นบริบทสำหรับการติดตาม ไม่ใช่คำแนะนำการลงทุน

### Stock / Symbol Context

- หุ้นตัวนี้มี momentum ดีขึ้นเมื่อเทียบกับช่วงก่อนหน้า
- ราคาเข้าใกล้โซนที่ควรเฝ้าดูความเสี่ยงและแรงขาย
- ปริมาณซื้อขายเพิ่มขึ้น จึงควรตรวจว่ามีข่าวหรือปัจจัยเฉพาะตัวรองรับหรือไม่
- valuation ยังต้องเทียบกับค่าเฉลี่ยอุตสาหกรรมและแนวโน้มกำไร
- ควรดูงบการเงิน กระแสเงินสด หนี้สิน และ margin ประกอบก่อนสรุปมุมมอง

### Risk Language

- ความเสี่ยงหลักคือข่าวกลับทิศ, volume ไม่ยืนยัน, และ market sentiment อ่อนลง
- ถ้าราคาหลุดโซนสำคัญ ควรทบทวนสมมติฐานเดิม
- จุดที่ต้องระวังคือ downside, สภาพคล่อง, และความผันผวนของตลาดรวม
- ข้อมูลนี้ช่วยตั้งคำถามก่อนตัดสินใจ ไม่ได้แทนการวิเคราะห์ส่วนตัว
- ควรกำหนดแผนจัดการความเสี่ยงก่อนเพิ่ม exposure

### Suggested Checks

- ตรวจ volume ว่ามากกว่าค่าเฉลี่ยหรือไม่
- เทียบกับ sector peers เพื่อดูว่าเป็นแรงเฉพาะตัวหรือแรงทั้งกลุ่ม
- ตรวจข่าวล่าสุด งบการเงิน และปฏิทินประกาศผลประกอบการ
- ดูแนวรับ แนวต้าน และระดับที่ thesis เดิมควรถูกทบทวน
- บันทึกเหตุผล สมมติฐาน และเงื่อนไขที่ทำให้ต้องเปลี่ยนมุมมอง

### Portfolio / Journal

- การตัดสินใจนี้ควรถูกประเมินจาก process ไม่ใช่ผลลัพธ์ครั้งเดียว
- ข้อผิดพลาดที่ควรทบทวนคือ position sizing, timing, และการยอมรับความเสี่ยง
- journal ควรแยกแผนก่อนเข้า, สิ่งที่เกิดขึ้นจริง, และบทเรียนหลังปิดสถานะ
- พอร์ตควรกระจายความเสี่ยงตามเป้าหมาย เวลา และ drawdown ที่รับได้
- ถ้าเหตุผลในการถือเปลี่ยนไป ควรบันทึกและทบทวน thesis ใหม่

### Alerts / Watchlist

- ตั้ง alert เพื่อกลับมาตรวจสอบ ไม่ใช่เพื่อสั่งซื้อขายอัตโนมัติ
- watchlist คือรายการติดตามเพื่อรอข้อมูลยืนยัน
- alert นี้ชี้ว่าราคาแตะเงื่อนไขที่กำหนดไว้ ควรตรวจบริบทก่อนตัดสินใจ
- สัญญาณจากราคาอย่างเดียวไม่พอ ควรตรวจ volume, ข่าว และตลาดรวม

### AI / Agent Copy

- มุมมองนี้สร้างจากข้อมูลที่ระบบมีในขณะนี้
- สมมติฐานสำคัญคือข้อมูลราคาและข่าวล่าสุดสะท้อนสถานการณ์จริงพอสมควร
- สิ่งที่ AI ยังไม่รู้คือข้อมูลภายในบริษัท, รายการซื้อขายจริงของผู้ใช้, และข่าวที่ยังไม่ปรากฏในแหล่งข้อมูล
- ใช้คำตอบนี้เป็น checklist สำหรับตรวจต่อ ไม่ใช่คำสั่งลงทุน
- ถ้าข้อมูลใหม่เปลี่ยน คำตอบนี้ควรถูกทบทวนใหม่

## Safe English Phrases

Use sparingly. Thai should be primary.

- Market context
- Risk checklist
- Momentum context
- Valuation check
- Evidence to verify
- Assumptions to revisit
- Scenario, not forecast
- Decision support only
- Not investment advice

## Replace Unsafe Wording

| Avoid | Use Instead |
|---|---|
| ซื้อเลย | ควรตรวจปัจจัยยืนยันก่อนตัดสินใจ |
| ขายทันที | ควรทบทวน thesis และความเสี่ยง |
| สัญญาณซื้อ | momentum เริ่มฟื้น ต้องตรวจ volume และแนวโน้มหลัก |
| สัญญาณขาย | momentum เริ่มอ่อน ต้องตรวจแนวรับและปัจจัยข่าว |
| หุ้นน่าซื้อ | หุ้นที่ควรติดตามต่อ |
| หุ้นเด็ด | หุ้นที่มีประเด็นให้ตรวจสอบ |
| ราคาเป้าหมายที่ควรซื้อ | โซนราคาที่ควรใช้ทบทวนสมมติฐาน |
| กำไรแน่นอน | มี upside scenario แต่ยังมีความเสี่ยง |
| AI แนะนำให้ซื้อ | AI สรุปบริบทและสิ่งที่ควรตรวจต่อ |
| ต้องเข้า | ยังไม่ควรสรุปโดยไม่มีข้อมูลยืนยัน |

## Required Disclaimer Patterns

Short:

```text
ข้อมูลนี้ใช้เพื่อการศึกษาและการวิจัยเท่านั้น ไม่ใช่คำแนะนำการลงทุน
```

Standard:

```text
ข้อมูลนี้มีวัตถุประสงค์เพื่อการศึกษาและการวิจัยเท่านั้น ไม่ใช่คำแนะนำการลงทุน ผู้ใช้ต้องรับผิดชอบต่อการตัดสินใจลงทุนของตนเอง
```

Agent / War Room:

```text
ผลลัพธ์นี้เป็น decision support เพื่อช่วยตั้งคำถามและตรวจความเสี่ยง ไม่ใช่คำสั่งซื้อขายหรือคำแนะนำการลงทุน
```

## Output Template

For generic finance AI answers, use this order:

1. ข้อมูลที่ใช้
2. สมมติฐาน
3. สิ่งที่เห็น
4. ความเสี่ยง
5. สิ่งที่ควรตรวจต่อ
6. ข้อจำกัดความรับผิด

## Hard Stop

If requested copy would tell the user to buy, sell, hold, allocate, or guarantee return, rewrite it into research/checklist language.
